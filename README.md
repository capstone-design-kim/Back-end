### 🏷️ 작업중인 브랜치가 master인 경우
      > git branch -M main을 실행해 branch의 이름을 master에서 main으로 변경  
      
### 🏷️ git push를 했는데  ! [rejected]  main -> main (fetch first) 에러 발생한 경우
      >  git pull --rebase origin main으로 로컬과 원격 저장소를 동기화 한 후에 push 진행
